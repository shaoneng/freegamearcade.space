Run

How to Play Run

Run is a fast-paced platformer where you guide a small alien sprinting through a 3D tubular course suspended in space. The primary goal is to survive as long as possible by navigating an endless tunnel filled with increasingly difficult gaps.

The game's standout feature is its unique gravity-shifting mechanic. Whenever you run onto a wall or the ceiling, the gravity reorients, allowing you to stick to that surface. Use this to your advantage to find a safe path and avoid falling into the void. Use the arrow keys to move left and right, and the spacebar to jump.

As you progress, the gaps in the track become wider and more frequent, demanding quick reflexes and precise timing. A single misstep will send you tumbling into space, forcing you to restart the current level. You can choose between Adventure Mode to progress through levels sequentially or Infinite Mode to test your endurance.

Originally a Flash classic created by Joseph Cloutier, Run has been rebuilt in HTML5 for modern browsers. If you master the gravity-bending gameplay, be sure to check out Run 3D for even more levels, new obstacles, and expanded mechanics.
